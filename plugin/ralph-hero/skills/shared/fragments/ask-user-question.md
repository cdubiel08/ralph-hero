## AskUserQuestion Convention

Options must be **self-contained** — the user should be able to choose without scrolling back to surrounding context.

- **Label** = action verb + concrete target (max ~5 words). Examples: "Merge PR #627", "Review Plan #597", "Clean Board"
- **Description** = what the target is + what happens when selected. Examples: "ralph-playwright (4 phases) — runs code review, checks CI, merges if clean"

Do NOT use bare numbers ("1", "2", "3") or generic labels ("Option A") as labels. The label field is prime screen real estate — use it to communicate the action.
